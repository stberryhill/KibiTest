## About
KibiTest was written as a *dead simple* testing solution for my other C projects, which are starting to multiply for a couple reasons:
* As an experiment I started using C as my hobby language.
* I've discovered the simplicity of the language makes me very productive, because I don't get bogged down with a million different ways to do something.
  
  
## KibiTest in the Wild
Currently KibiTest is being used to test the terminal manipulation library I've been making, [Bulwark](https://github.com/stberryhill/Bulwark).
